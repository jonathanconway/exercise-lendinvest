# LendInvest Exercise

Application customers use to manage their loans.

## Pre-requisites

* NPM

## Checkout & Install

```bash
git clone https://github.com/jonathanconway/exercise-lendinvest.git
cd ./exercise-lendinvest
npm install
```

## Run

```bash
npm start
```

## Test

```bash
npm test
```

## Assumptions

* `term_remaining` values are in seconds
